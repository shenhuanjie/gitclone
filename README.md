# OpenSpec Project

A specification-driven API project using [OpenSpec](https://openspec.dev).

## Getting Started

### Prerequisites

- Node.js 18+
- pnpm 8+

### Installation

```bash
pnpm install
```

### Development

```bash
# Start development mode
pnpm dev

# Build specifications
pnpm build

# Validate specifications
pnpm validate
```

## Project Structure

```
openspec/
├── config.yaml          # OpenSpec configuration
├── specs/               # API specifications (OpenAPI, AsyncAPI, etc.)
└── changes/             # Changelog and version history
```
